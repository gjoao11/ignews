## Instalação

> Adicione as variáveis de ambiente em um arquivo `.env.local`. Veja o `.env.local.example`.

```bash
  # Clone o repositório
  $ https://github.com/gjoao11/ignews.git

  # Acesse o diretório do app
  $ cd ignews

  # Instale as dependências
  $ yarn

  #Inicie a aplicação
  $ yarn dev

  # acesse `http://localhost:3000`
```
